Detailed Description

This class calculates low-resolution elevations.

This class estimates the low-res elevation at a point by finding the kth-smallest elevation value for a block of elevation data centered at the point. The size of the tile is set to be nx-by-ny pixels and the class is passed a block of data of that size The kth-smallest value has been hard-wired to be 20%.
