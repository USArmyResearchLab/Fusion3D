Detailed Description

Stores the set of files in a given directory that match a given pattern and, if desired, that were last written after a given time.

Stores the set of files in a given directory that match a given pattern and, if desired, that were last written after a given time. This class can be used for realtime operation where data generated by sensors can be input to the viewer in near realtime. It is a simpler way of getting realtime inputs than streaming. The class can search arbitrarily deep in a directory tree.
