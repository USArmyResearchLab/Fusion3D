Detailed Description

Reads point-cloud lidar data in .bpf format.

Reads point-cloud lidar data in .bpf format.
This class has the capability to clip the point cloud to a specified region in x and y (not z).
Most of the work is done by helper class image_bpf_class that does the actual reads. This is the most straightforward use of image_bpf_class and may not be most efficient.
