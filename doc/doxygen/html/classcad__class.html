Detailed Description

Imports and exports files in CAD formats.

Implemented using the assimp library which is a uniform interface into a number of different CAD formats including .obj and collada.

◆ canHandleThisFormat()

bool cad_class::canHandleThisFormat ( string  filename)

Returns true if this class can handle the format associated with the input filename, false if not.

This method checks for the most common formats read by Assimp and returns true for these. If format is less common, it must use an Assimp importer object which takes some time, so may be relatively slow.

Write a file – Not implemented.
