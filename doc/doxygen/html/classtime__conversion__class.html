Detailed Description

Does format conversions for time and date data.

Converts time data between character, integer and floating point formats. There are many different character formats – this includes several that have been encountered, but probably not all of them.

◆ set_char()

int time_conversion_class::set_char ( const char *  tchar)

Set time in a character format.

Recognizes formats (in order of search):
yyyy-mm-dd T hh:mm:ss.sssZ
(iff if finds the char string ' T ')
yyyy-mm-ddThh:mm:ss.sssZ
(iff if finds the char 'T')
yyyy-mm-dd hh:mm:ss
(iff 19 chars)
hh:mm:ss / hh.mm.ss / hh:mm:s / hh.mm.s
(iff 7 or 8 chars)
trailing blanks are stripped prior – blanks may come from .dbf files
