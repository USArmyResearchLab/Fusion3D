Detailed Description

Virtual parent class of all classes that import or export in vector format.

Object types this class can handle
kml using libkml library points, lines, multilines, polygons, mutlipolygons, symbols, latlonbox
shp using GDAL library points line, no name
shp using my code LOS outlines
osm using GDAL library points with names, lines
csv using my code GPS track from LORD
sbet using my code GPS track from Applanix

Flags
format_flag
0 = unknown
1 = KML
2 = shapefile
3 = csv
4 = txt My special .csv file
5 = osm OpenStreetMap

Implemented for only single values for color, altitude mode, and polygon fill mode per file.
These would be set on write/read and are overridden if values explicitly set in the file.
Modified values can then be fetched.
