Detailed Description

Resamples an image with translation, rotation and scaling.

The input image is unsigned char and the output image in floating point. Memory for input and output images is allocated outside of this class.

Coordinate system:
Origin @ center of image
x-axis toward right
y-axis down
rotation positive clockwise (x toward y)
order: (1) scale (2) rotate (3) translate
