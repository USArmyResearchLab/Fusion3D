Detailed Description

Creates a track with associated locations, times and velocities.

This class was written to import location data (like from a track that you digitize yourself) and add time data to make a reasonable time track.

◆ distance_average()

int time_track_class::distance_average ( )

Calculates a smoothed path and look direction az.

First calculates the az at each point (average of forward and backward az). Then averages all track points within radius avgRadialDist.
