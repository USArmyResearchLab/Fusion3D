Detailed Description

Maintains the central index for vector overlay files.

Allows filenames and pathnames for vector overlay file to be fetched that meet certain criteria. These criteria include the type of data, and the location (the name of the map they are associated with).

type_flags
0 = "map" – KML from DEM fileset – display but do not write
1 = "display" – display only, cant edit or animate
2 = "bookmarks" – bookmarks (can edit, jump to) 3 = "mti" – GMTI – adds special processing for GMTI data
4 = "track" – adds special processing for time/motion and measuring distance
5 = "sensorml" – adds special processing for ugs SensorML 6 = "om" – adds special processing for ugs OM 7 = 'draw" -- files for drawing, typically military scenarios 8 = 'cad" – files for CAD models 99 = deleted
vis_flags 1 if file is currently visible, 0 if invisible.

update_flags Per type, 1 if all overlays of that type need update, 0 if not.
