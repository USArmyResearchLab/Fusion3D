Detailed Description

Adds OpenInventor functionality to cad_manager_class.

Adds capability to draw the objects that the class imports. Currently implemented to draw
individual point features from KML
point features making up a line such as target tracks from KML/shapefile/csv/txt files
complex features such as polylines/polygons from KML or shapefiles (LOS boundaries).
