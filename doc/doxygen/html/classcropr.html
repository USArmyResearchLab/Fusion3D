Detailed Description

Manages camera movement.

Crops a single point cloud file to specified subregion.

The main menu allows the user to browse for a point-cloud file in either .las or .bpf format. When the file is chosen, the limits of the point-cloud data are displayed in the menu and can be modified by the user.
