Detailed Description

Base class for many classes.

Includes a reference location – latitude, longitude , UTM coordinates and zone and reference name (e.g. "Lubbock") and methods to set this location. Includes basic methods to display basic warning and exit messages. Includes method to get the name of the class (e.g. "KML"
