Detailed Description

Child class of SoQtFullViewer that limits the OpenInventor viewer to things that are reasonable for viewing terrain.

The class was based on FullViewer rather than ExaminerViewer because the latter does not allow delayed build of the widget and therefore deleting the viewer nav buttons on the right of the window.

This viewer restricts the panning options so that the user always has heads up (the ground tilts forward, but not side-to-side) and so that the ground can't tilt backward or further forward than a few degrees above the horizon (one cant go under the ground).

The viewer also deletes some of the right-hand buttons that can get the user in trouble.

Also the viewer hides the differences between parent ExaminerViewer classes so that it can be passed to other classes without modifying code.
