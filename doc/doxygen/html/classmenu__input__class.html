Detailed Description

Manages main menus at the top of the Fusion3D main window.

Simple menu items, like printing out 'About' messages, are implemented from within this class. More complex menu items – those requiring additional menus – are implemented using additional classes. Names for these classes start with 'dialog' then a designator for the main menu they fall under (like 'Vec' for vector) and finally the submenu (like 'Display' for toggle display).
