Detailed Description

Does coordinate transformations SGI GL style.

Coordinate transformations

Author: J.F. Dammann
Revision 2.0
Date: 94/03/11

Transformations are SGI GL style. Homogeneous coordinates are used and a single 4x4 transformation matrix is maintained. Compositing matrices is done by pre-multiplying the new matrix onto the existing matrix:

[ new ] [ pre-existing] = [ result ]

This matrix is used to generate a new point (x',y',z') by

[x,y,z,w] = [ 4x4 matrix[row][col] ] = [x',y',z',w']

xt = matrix[0][0]*x[i] + matrix[1][0]*y[i] + matrix[2][0]*z[i] + matrix[3][0];
yt = matrix[0][1]*x[i] + matrix[1][1]*y[i] + matrix[2][1]*z[i] + matrix[3][1];
zt = matrix[0][2]*x[i] + matrix[1][2]*y[i] + matrix[2][2]*z[i] + matrix[3][2];

See Appendix C of Graphics Library Programming Guide for trans. matrices

Notes
scale, translate and rotate should all composit properly
The SGI matrix stack has not yet been implemented.
