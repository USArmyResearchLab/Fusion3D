Detailed Description

Defines a group of user buttons on the left of the main graphics window – does the operating - system independent operations.

The buttons are a limited user interface that is an integral part of the original OpenInventor interface (as opposed to the menus which were added later and operate outside the OpenInventor environment).
Whenever a button is added or removed, the entire column of buttons is deleted then regenerated. Each button is controlled by a flag in parm button_vis_flags– 0 to not include button, 1 to include button
100 vis flags are used – only some are used and others as backup.

Buttons are turned on/off either from a project file (parsed within this class) or from menus using the IOV Global GL_button_mod

int *button_vis_flag; Array of flags that govern whether or not each button is currently visible
0 - 19 default buttons
0 - Map vis
1 - Vector vis
2 - Point cloud vis
3 - Bookmark vis
4 - track / drawing vis
5 - NorthUp vis
6 - Tile(toggle draping) vis
7 - A1 - DEM vis
8 - Highlighting of obstructions(points where a2 - DEM significantly higher than a1 - DEM) vis
9 - Hi - res texture vis
10 - Reset vis
20 - Script play
21 - Script next
30 - Clock Reverse 1
31 - Clock Reverse
32 - Clock Stop
33 - Clock Forward
34 - Clock Forward 1
70 - 73 - Jump - to - bookmark
80 - Editing off
81 - Add point or add point to end of line
82 - Move point – point closest to mouse - press to mouse - release
83 - Del point – point closest to mouse press
84 - New line solid
85 - New line dashed
86 - New point
