Detailed Description

Manages point clouds.

Manages reading, writing and displaying point clouds. The class is primarily used for terrain mapping point clouds – sensing terrain from the air. Experiments were also done to demonstrate merge of airborne and mobile mapping lidar but this feature has not been thoroughly developed or tested.
