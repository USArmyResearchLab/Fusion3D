Detailed Description

This class reads, filters and writes lidar point cloud files in .bpf format.

It was provided by M. Brandon Cox - Booz Allen Hamilton - cox_m.nosp@m.icha.nosp@m.el@ba.nosp@m.h.co.nosp@m.m.
